== Description ==

2025 Bulma is a modern WordPress theme that seamlessly integrates the powerful Bulma CSS framework with the robust foundation of Twenty Twenty-Five. This theme combines the simplicity and accessibility of WordPress's default theme with the utility-first approach and modern design system of Bulma.

**Key Features:**

🎨 **Bulma CSS Framework Integration**
- Full integration of Bulma 1.0.0+ CSS framework
- Utility-first CSS classes for rapid development
- Responsive grid system and flexbox utilities
- Modern component library with buttons, cards, modals, and more

🚀 **Enhanced Styling Capabilities**
- Custom Bulma-based button styles with multiple variants
- Advanced component styling for forms, navigation, and layouts
- Editor-specific styles for seamless block editor experience
- Professional color schemes and typography options

📱 **Fully Responsive Design**
- Mobile-first responsive design approach
- Optimized for all screen sizes and devices
- Touch-friendly interface elements
- Cross-browser compatibility

🎯 **Rich Pattern Collection**
- 80+ professionally designed block patterns
- Page templates for business, portfolio, blog, and e-commerce
- Header and footer variations
- Hero sections, CTAs, testimonials, and pricing tables
- Specialized patterns for post formats and landing pages

⚡ **Performance Optimized**

- Optimized for fast page loading speeds
- SEO-friendly markup structure

🌍 **Accessibility Ready**

== Installation ==

1. Upload the `twentytwentyfive-bulma` folder to the `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Customize the theme appearance in Appearance > Customize
4. Explore and use the available block patterns in the block editor

== Customization ==

The theme provides several customization options:

1. **Bulma Utilities**: Use Bulma's utility classes directly in your content
2. **Custom Templates**: Special page templates for buttons and components demo
3. **Style Variations**: Multiple pre-designed style options
4. **Color Palettes**: Extensive color customization through theme.json

To use Bulma classes, simply add them to your blocks in the editor:
- Buttons: `button is-primary`, `button is-info is-large`
- Layout: `columns`, `column is-half`
- Components: `card`, `navbar`, `menu`

== Demo Templates ==

The theme includes two demonstration templates:

1. **Bulma Buttons Demo** (`template-buttons-demo.php`)
   - Showcases all Bulma button styles and variants
   - Includes size variations and color schemes

2. **Bulma Components Demo** (`template-components-demo.php`)
   - Demonstrates Bulma components integration
   - Features cards, forms, modals, and navigation elements

== Frequently Asked Questions ==

= How do I use Bulma classes in the block editor? =
You can add Bulma utility classes to most blocks by using the "Additional CSS Class(es)" field in the block settings.

= Can I use this theme with page builders? =
Yes, 2025 Bulma works well with most page builders, but it's optimized for the WordPress block editor.

= Is Bulma loaded on the frontend and backend? =
Yes, Bulma styles are loaded both in the site editor and on the frontend for consistent styling.

= How do I customize the colors? =
Colors can be customized through the WordPress customizer or by modifying the theme.json file.
